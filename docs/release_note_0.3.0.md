# Dictyy v0.3.0 Release Notes

## 新功能

- **屏幕取词**：选中任意应用中的英文单词，自动弹出气泡显示释义摘要
- **气泡窗口**：显示单词、音标、简要释义，点击「详细 →」跳转到主界面查看完整内容
- **单词摘要预加载**：启动时将所有单词摘要（约 5 万条）加载到内存，气泡查询瞬时响应

## 改进

- **词形还原**：支持复数、过去式、进行时等词形变化的自动还原（如 resources → resource、removed → remove）
- **日志持久化**：调试日志写入 `%LOCALAPPDATA%\Dictyy\debug.log`，便于问题排查
- **气泡位置优化**：根据选中文本位置自动计算气泡显示位置，避免超出屏幕边界

## 技术实现

- 屏幕取词采用 UI Automation API 轮询方案（200ms 间隔）
- 选中文本稳定 500ms 后显示气泡，避免频繁弹出
- 气泡与主窗口通过 Tauri 事件机制通信
