# Dictyy v0.2.0 Release Notes

## 新功能

- **多词典支持**：新增柯林斯英汉双解词典、词根词缀词典、GPT4 缓存查询
- **Tab 切换视图**：查询结果按数据源分 Tab 展示，方便对比不同词典释义
- **搜索建议优化**：输入时查询所有词典，显示更全面的匹配结果
- **动态窗口尺寸**：窗口大小自适应屏幕（宽度 2/3，高度 3/4），居中显示

## 改进

- 简化 Status Bar，仅显示快捷键提示和版本号
- 移除 Esc 隐藏窗口功能，统一使用 `Ctrl+`` 切换
- 柯林斯词典按词性分组显示，PHRASE 和 See Also 单独分区
- 词根词缀相关词汇渲染优化

## 开发者

- 新增 `scripts/import_mdx.py` MDX 词典导入脚本
- 更新架构文档 `docs/design.md`
